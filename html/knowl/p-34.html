Paragraphe

Parmi les trois formats (HTML, PDF, imprimé), le HTML est optimal pour un affichage en classe si vous disposez d'un projecteur adapté. Le HTML est également le meilleur pour la navigation, car les liens vers des références internes et externes sont beaucoup plus évidents. Nous vous recommandons d'enregistrer une version téléchargée du format PDF en tant que sauvegarde au cas où vous n'auriez pas accès à Internet. C'est une bonne idée pour chaque étudiant d'avoir une version imprimée du cahier d'activités, que vous pouvez acquérir sous forme de document PDF sur demande directe à l'auteur. De nombreux enseignants utilisent le PDF pour imprimer des modules de cours que les étudiants peuvent acheter dans leur librairie locale.
