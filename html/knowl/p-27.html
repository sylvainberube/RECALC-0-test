Paragraphe

Le texte est disponible dans trois formats différents : HTML, PDF et imprimé, chacun étant disponible via des liens sur la page de destination à l'adresse https://activecalculus.org /. Les deux premiers formats sont gratuits. Si vous envisagez d'utiliser le livre électroniquement, le meilleur mode est le HTML. La version HTML a fière allure dans n'importe quel navigateur, y compris sur un smartphone, et les liens sont beaucoup plus faciles à naviguer en HTML qu'en PDF. Quelques suggestions directes particulières sur l'utilisation du HTML suivent parmi les prochains paragraphes. Alternativement, vous pouvez regarder cette courte vidéo de l'auteur (basée sur l'utilisation du texte Active Calculus, qui est similaire). Il est également judicieux de télécharger et d'enregistrer le PDF, car vous pouvez utiliser le PDF hors ligne, tandis que la version HTML nécessite une connexion Internet. Une copie papier bon marché est disponible sur Amazon.
