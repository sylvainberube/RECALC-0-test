Paragraphe

Le code source PreTeXt du texte se trouve sur GitHub. Si vous trouvez des erreurs dans le texte ou avez d'autres suggestions, vous pouvez signaler un problème sur GitHub ou envoyez directement un courriel à l'auteur. Pour dialoguer avec les enseignants qui utilisent le texte, nous maintenons à la fois une liste de diffusion et le Open Calculus blog : vous pouvez demander que votre adresse soit ajoutée à la liste de diffusion en contactant l'auteur. Enfin, si vous êtes intéressé par une présentation vidéo sur l'utilisation du texte similaire Active Calculus, vous pouvez voir ce présentation vidéo en ligne au Séminaire électronique du MIT sur l'enseignement des mathématiques : à environ 17 minutes, la partie commence où nous démontrons les caractéristiques et comment utiliser le texte.
